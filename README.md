# django-dashboarder
Django app to beautify and add more functions to admin interface.

## Local Usage

This package is no longer available on PyPI. To use it locally, copy the `dashboarder` directory into the root of your Django project (alongside `manage.py`) and add it to `INSTALLED_APPS` in your settings:

```python
INSTALLED_APPS = [
    ...
    'dashboarder',
]
```
